# n8n 学习路线图

## 🗺️ 学习概览

本路线图将帮助您**系统性地掌握 n8n**，从零基础到能够独立构建复杂的自动化解决方案。

**预计学习时间**：8-12 周（每天 1-2 小时）
**学习方式**：理论 + 实践 + 项目实战

## 🌱 第一阶段：基础入门（第 1-2 周）

### 📚 理论学习
- [x] **什么是 n8n** - 理解核心概念和基本特性
- [x] **低代码平台特点** - 掌握低代码开发思维
- [x] **AI 时代自动化思维** - 了解 AI 与自动化的结合

### 🛠️ 环境搭建
- [ ] **本地安装指南** - 在本地环境安装 n8n
- [ ] **Docker 部署** - 使用 Docker 快速启动
- [ ] **云服务器部署** - 在生产环境部署 n8n

### 🎯 第一个工作流
- [ ] **Webhook 触发** - 创建第一个自动化流程
- [ ] **HTTP 请求节点** - 学会调用外部 API
- [ ] **数据转换** - 掌握数据格式转换技巧

### 📋 学习目标
- 理解 n8n 的基本概念和工作原理
- 能够独立部署 n8n 环境
- 创建简单的自动化工作流
- 掌握基本的节点配置方法

## 🚀 第二阶段：核心技能（第 3-4 周）

### ⚙️ 操作类节点

### 🔄 流程控制
- [ ] **条件判断** - IF 节点和 Switch 节点
- [ ] **循环处理** - 处理数组和批量数据
- [ ] **错误处理** - 异常情况的处理机制

### 📊 数据处理
- [ ] **数据映射** - 节点间数据传递
- [ ] **数据验证** - 确保数据质量
- [ ] **数据转换** - 格式转换和清洗

### 📋 学习目标
- 熟练掌握各类节点的使用方法
- 能够设计复杂的数据流转逻辑
- 掌握错误处理和调试技巧
- 理解工作流的执行机制
