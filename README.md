# n8n 中文学习教程 🚀

> **前景提要**：这份教程专门为小白选手、非程序员准备的系统学习教程。如果你有一定基础，或者只是想查某个节点的操作，不用跟着这套流程，跳着阅读就行。

> 
> 记得踩在前人肩膀上才能走更高，推荐这个仓库：[awesome-n8n-templates](https://github.com/enescingoz/awesome-n8n-templates) 参考改造或创作属于你自己的工作流。

> 💡 **学习重点**：不要追求一步到位。先让流程"跑通"，再逐步优化。

> 一个完整的 n8n 工作流自动化学习教程，帮你从零开始掌握 n8n，构建智能自动化解决方案

## 🎯 为什么选择 n8n？

### 🔗 强大的集成能力
- **1000+ 第三方服务**：Google Sheets、Notion、飞书、企业微信等
- **半封装模式**：既可以用现成节点，也能插入 JavaScript/Python 代码
- **BYOK 支持**：用自己的 OpenAI API 密钥，灵活控制成本

### 🤖 AI 友好设计
- **内置 LangChain 节点**：轻松集成 GPT、通义千问等大模型
- **可视化 AI 编排**：不用写代码就能让 AI 参与决策和处理
- **记忆与工具链**：支持 AI Agent 的复杂推理流程

### 💻 适合个人和小团队
- **私有化部署**：数据完全由你掌控，更安全
- **"寄生系统"**：为现有系统"打补丁"或"加外挂"
- **快速迭代**：先让流程跑通，再逐步优化

## 🔮 学习心法（必读）

### 💡 核心原则
1. **不要追求"完全理解"**  
   → 很多配置（如 OAuth）你可能不懂原理，但只要"work"了就先用着。

2. **善用搜索引擎和 AI**  
   → 报错信息 → 截图 → 丢给 ChatGPT："这个 n8n 错误怎么解决？"

3. **从小处着手，快速迭代**  
   → 先做"最小可用流程"，再逐步加功能。

4. **把 n8n 当"寄生系统"**  
   → 不改变现有业务，只在上面加一层自动化"外挂"。

5. **接受不完美**  
   → 低代码的稳定性不如工程系统，但胜在**灵活、快速、自由**。

### 🧭 学习建议
- **目标明确**：知道要学什么，学到什么程度
- **循序渐进**：按照教程顺序学习
- **动手实践**：理论结合实践，学以致用
- **社区交流**：与他人分享，共同进步
